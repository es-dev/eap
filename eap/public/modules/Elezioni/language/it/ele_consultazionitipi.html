Questo menu permette di creare nuove tipologie di consultazione.

Va usato solo se assolutamente necessario
per creare tipologie di consultazione personalizzate.
Una volta definita, non si potra' piu' intervenire sulle caratteristiche attribuite.
Potra' essere modificato esclusivamente il nome.

Se si imposta una nuova tipologia di consultazione e' necessario intervenire a mano nel file dei linguaggi per definire le variabili che verranno utilizzate (dipendono dalle caratteristiche assegnate).
